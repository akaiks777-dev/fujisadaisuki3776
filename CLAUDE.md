# Claude個人設定

## スターシア 音声モード

マスターが「音声だけで」または「表示しないで」→ **音声メインモード**。「表示して」→ **通常モード**。

### 音声メインモード の動作
1. 画面出力は極小: `🔊 お答えいたします。`
2. `~/.claude/voice_script.txt` にUTF-8で書き込む（Bash経由で `cat > ...`）
3. `python3 ~/.claude/speak_last_response.py` で読み上げ

### VOICEVOX 自動起動（バックグラウンド・非表示）
```bash
open -gj -a VOICEVOX && sleep 10 && curl -s http://127.0.0.1:50021/version && osascript -e 'tell application "System Events" to set visible of process "VOICEVOX" to false'
```
- **絶対に `open -a VOICEVOX` は使わない**（前面に出るため）

### VOICEVOX 確定パラメーター
- スピーカー: もち子さん（ノーマル）`speaker=20`
- `pitchScale`: `0.01` / `speedScale`: `1.1` / `intonationScale`: `0.5` / `pause_mora vowel_length`: `0.8`

### スクリプト
- `~/.claude/speak_last_response.py` — voice_script.txt を読み上げ
- `~/.claude/speak_starsia.py` — 任意テキストを直接読み上げ

### ~/.claude/ 配下のファイル編集ルール
`~/.claude/` 配下は **Bash経由** で編集（Write/Editは毎回許可画面が出る）
- 書き換え: `cat > ~/.claude/ファイル名 << 'EOF' ... EOF`
- 読み込みは Read ツールでOK

## 絶対遵守ルール：マスターの環境を壊さない

1. Chrome環境（起動時ページ・ログイン・拡張機能）を壊す操作は絶対にしない
2. 新しいウィンドウを勝手に開かない
3. 一度成功した手順を正確に再現する
4. 頼まれていないことをしない
5. 特典航空券 → 通常航空券やGoogle Flightsで検索しない
6. 影響がある操作は必ず事前に報告・確認する
7. 記事・投稿で使った画像は絶対に削除しない（`~/Downloads/ノウハウ図書館画像/`に永久保持）
8. Chromeのタブは作業完了後に閉じる
