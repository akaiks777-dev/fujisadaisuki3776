#### 新しい本文（全文）
```
みなさん、スクショってどうやって撮ってます？

Command + Shift + 3で全画面、Command + Shift + 4で範囲選択。便利なショートカットですよね。

……私には無理でした。

62歳、指3本同時押しがどうしてもできない。覚えたつもりで翌日には忘れてる。メモに書いても、そのメモがどこにあるかわからない。もうね、自分が嫌になりますよ😭

で、ある日Claude Codeに愚痴ったんです。

「スクショ撮りたいんだけど、ショートカットキー覚えられないんだよね…もっと簡単にならない？」

そしたらClaudeが「フック機能を使えば、好きな言葉を打つだけで自動でスクショ撮れますよ」って。

えっ、そんなことできるの？

■「はる」が生まれた瞬間

「じゃあ何て打ったら動くようにする？」って聞かれて、考えたのが「はる」でした。

スクショって撮ったら画面に「貼る」でしょ？だから「はる」。

2文字で打ちやすいし、やることそのまんまだし。「スクショ自動化」なんて難しい名前より、「はる」のほうが62歳の私にはしっくりくる。

Claudeに「"はる"って打つだけでスクショ撮れるようにして」って言ったら、ものの数分で作ってくれました。

試しに「はる」って打ってみたら……

パシャッ。

撮れた。ほんとに撮れた。しかもClaudeが勝手に画像を読み込んで、画面に何が映ってるか理解してくれる。

「おお！！」って声出ましたよ、一人で😆

■ 調子に乗ってさらに注文

うまくいくと欲が出るもので。

「ねえ、撮ったスクショって気づくとデスクトップに溜まるんだけど、自動で消してくれない？」

Claudeは嫌な顔ひとつせず（AIだから当たり前だけど）、セッション終了時に自動でゴミ箱に入れてくれる仕組みも追加してくれました。

つまり今、私がやることは「はる」と打つだけ。

撮影→AI読み込み→後片付け、全部おまかせ。

■ 正直に言います

この仕組み、私は1行もプログラムを書いてません。

やったことは3つだけ：

1.「スクショ簡単に撮りたい」って愚痴った
2.「"はる"で動くようにして」って注文した
3.「ゴミも自動で消して」って追加注文した

全部、日本語でお願いしただけ。これがClaude Codeの「フック機能」というやつらしいです。名前は難しそうだけど、やることは「お願いするだけ」。

■ ちょっとだけ注意

この方法、画面をそのまま撮影するので、見せたいウィンドウの手前に別の画面があると、そっちが写っちゃいます。

「はる」って打つ前に、見せたい画面が一番手前にあるか確認してくださいね。

あと、おすすめはデスクトップPC＋大きいモニターの環境です。画面が大きいほどAIが読み取れる情報量が増えて、精度が上がります。私はMac miniに4Kモニター2枚で使ってますが、めちゃくちゃ快適です。

■ まとめ

62歳、パソコン苦手。ショートカットキーは何回覚えても忘れる。

でも「はる」は忘れない。だって2文字だもん。

難しいことはわからなくていい。「こうしたい」って伝えるだけで、AIが全部やってくれる。

楽したいから始めたことが、一番うまくいった。

同じように「パソコン苦手だけど何かやってみたい」って方、まずはAIに愚痴ることから始めてみませんか？意外となんとかなりますよ😊
```
