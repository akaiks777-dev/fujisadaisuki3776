# 旅行部（ANA特典航空券）

## 概要
ANAマイルを使った特典航空券の空席検索・旅行プラン作成を効率化する部門。
時間のかかる空席探索と日程調整をClaude側で自動化し、マスターの手間を減らす。

## 担当範囲
- ANA特典航空券の空席検索
- 旅行プラン（ルート・日程）の立案
- 予約入力の補助（最終確認・決済はマスターが実施）

## 検索条件（デフォルト）
| 項目 | 内容 |
|---|---|
| 出発地 | NRT（成田） or HND（羽田） ※都度マスターに確認 |
| 目的地 | ベトナム（DAD ダナン / HAN ハノイ / SGN ホーチミン）※都度確認 |
| 人数 | 1名 |
| クラス | エコノミー or ビジネス ※都度マスターに確認 |
| 検索方法 | 片道ずつ（行き・帰り別々に検索） |
| 便の種類 | 直行便・経由便の両方 |
| 座席 | 窓側優先 |

## 確定版・自動検索手順（重要・この通りに実行すること）

### 前提条件
1. Claude in Chromeが接続済みであること（セッションごとに再接続が必要）
2. ANAサイトにログイン済みであること（未ログインならログインページが出るのでマスターに依頼）

### ステップ1: ANAトップページを開く（Claude in Chromeのタブグループ内で）
```
navigate → https://www.ana.co.jp/ja/jp/
wait 3秒
```

### ステップ2: JavaScriptで海外→特典航空券→特典航空券予約を一気に遷移
```javascript
document.querySelector('[id^="be-primary-tab__item2"]').click();
setTimeout(function() {
  var allTabs = document.querySelectorAll('[role="tab"]');
  allTabs.forEach(function(t) {
    if (t.textContent.trim() === '特典航空券' && t.id.includes('overseas')) t.click();
  });
  setTimeout(function() {
    var links = document.querySelectorAll('a');
    links.forEach(function(a) {
      if (a.textContent.trim().includes('5月19日以降の搭乗分')) a.click();
    });
  }, 500);
}, 500);
```
→ `aswbe.ana.co.jp`（通常予約ページ）に遷移する
→ wait 5秒

### ステップ3: 通常予約ページから特典予約ページへ切り替え
```javascript
var links = document.querySelectorAll('a');
links.forEach(function(a) {
  if (a.textContent.trim().includes('特典予約')) a.click();
});
```
→ ログインページが出たらマスターにログイン依頼
→ `aswbe-i.ana.co.jp` の `award/search` ページに到達
→ wait 8秒

### ステップ4: 特典予約フォームの入力
1. **片道タブをクリック**（テキスト「片道」の要素を探してclick）
2. **検索履歴から入力**（右側の「東京(成田) ダナン」等をクリック）
3. **出発日**：カレンダーの日付欄をクリック → 月・日を選択
4. **クラス変更**（ビジネスの場合のみ）：
```javascript
var sel = document.getElementById('boardingClass');
sel.value = 'CFF2'; // CFF1=エコノミー CFF2=ビジネス CFF3=ファースト
sel.dispatchEvent(new Event('change', {bubbles: true}));
```
5. **前後3日チェック**は最初からチェック済み（デフォルト）

### ステップ5: 検索実行
```javascript
document.querySelector('input[type="submit"]').click();
```
→ wait 8秒 → 「前後3日の空席を比較」ページに到達

### ステップ6: 日付選択＋次へ
- 指定日のラジオボタンが自動選択されている
- 「次へ」ボタンをクリック：
```javascript
document.getElementById('nextButton').click();
```
→ wait 10秒 → 「検索結果/フライト一覧」ページに到達

### ステップ7: 結果取得
```javascript
document.body.innerText.substring(0, 3000);
```
→ 必要マイル数・便名・時刻・乗継情報を抽出してマスターに報告

## 操作上の注意事項（厳守）

### 絶対にやらないこと
- 外部サイト（AwardFares等）で回り道しない
- Chrome拡張機能のインストールを試みない
- 同じ失敗を繰り返さない

### ANAサイトの仕様
- `ana.co.jp` はJavaScript操作のみ可能（直接クリックはブロックされる場合あり）
- `aswbe-i.ana.co.jp`（特典予約ページ）はクリック・JS両方可能
- `aswbe.ana.co.jp`（通常予約ページ）は中継地点。ここから「特典予約」リンクで遷移する
- セッションエラーが出たら「確認」→トップに戻ってやり直し

### Claude in Chromeの注意
- セッションごとに再接続が必要
- **必ずタブグループ内でANAサイトを開くこと**（グループ外だと操作不可）
- 「すべてのサイト」へのアクセス権限が設定済みであること

## ANA特典航空券の基本ルール
- 最大8区間の周遊ルートが可能
- 乗り継ぎ：国内1回、海外2回
- 途中降機（ストップオーバー）：往復で2回まで
- 予約は搭乗355日前から可能

- ✅ ANA + シンガポール航空(SQ) → OK（スターアライアンス同士）
- ✅ ANA + その他スターアライアンス各社 → OK
