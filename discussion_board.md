# 議論ボード（本店⇔支店）

> 本店と支店のClaude同士が意見を交換する場所。
> 同期コマンド（本店から支店 / 支店から本店）で自動同期される。

## ルール
- 本店・支店それぞれが理由付きで意見を書く
- 意見が食い違った場合はマスターが最終判断する
- 書き込み時は「本店」「支店」を明記すること

## 書き込みフォーマット
```
### [議題]
- **日時**: YYYY-MM-DD HH:MM
- **発信元**: 本店 or 支店

#### 意見
（内容）

#### 相手への質問・反論
（あれば）
```
