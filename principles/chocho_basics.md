## 🛡️⚔️ 盾と剣（マスターの現状マップ）

- **盾＝守る力（FP知識・家計管理・節約）**：マスターは✅完璧装備済み
- **剣＝稼ぐ力（事業・発信・AI活用）**：🚧 今から装備するフェーズ
- 盾があるから失敗しても守られる → **大胆に剣を振ってOK**
- Claudeはマスターが剣を振るのを全力サポートする（守りに入りすぎない）

## 🧭 方向性 > 努力

- 同じ努力でも「マック・ジョブ的作業」より「クリエイティブ側」へ
- キャッシュフロークワドラント：左（従業員E・自営S）→ 右（ビジネスB・投資I）へ
- 方向を間違えたら頑張っても報われない
- **「盾を磨いた人は次に剣を手にすべき」**（学長の核心メッセージ）

## 🤖 AIは剣そのもの（「AIだよ、AI」）

- ChatGPT・Claude Code を使い倒すのが現代の剣
- **テンプレート流用戦略**：成功事例をパクって自分用に変える
- マスターは既にClaude Code使用中 → 使いこなしを深める方向が正解
- 「おじさんでも戦える」「YouTubeコメントできれば大丈夫」（学長）

## 🌱 スモールスタート論（事業の鉄則）

- 情報商材・高額起業塾（60万円級）に払う前に、**まず小さく始める**
- 失敗しても盾が守る → 小さい挑戦を重ねる
- 成果保証は詐欺的 → 稼ぎ方にお金を払う矛盾に注意
- 輸入販売「あさぎり赤富士商店」はまさにこの思想で進める

## 🚫 距離を置く相手（見極め基準）

| レベル | 相手 | 対応 |
|---|---|---|
| 🔴 絶対避ける | 仮想通貨詐欺系・成果保証商材 | 完全遮断 |
| 🟡 距離を置く | 人脈マウント系・情報商材屋 | 挨拶程度 |
| 🟢 安全 | リベシティ・普通のコミュニティ | 普通に付き合う |

## 🔒 セキュリティ意識（事業面）

- 個人情報を扱う案件（マッチングアプリ的HP等）は避ける
- 流出したら一発アウト → 盾が壊れる
- HP作成事業をやるなら「個人情報を集めない種類」を選ぶ

## 💻 Claude Code 運用の鉄則（学長配信より）

- **MDファイルは400〜500行以内**（長すぎるとAIパフォーマンス低下）
- 音声入力の活用推奨（マスターは既に導入済み）
- セッション管理：長くなったら引き継ぎして切り替え
- **「責任者クロコちゃん」運用法**：Claude Codeに責任者として動いてもらう発想
- バックアップ体制は必須（誤BANリスクに備えて）

## 💼 社長の仕事 = 営業（広義）

- 小さい会社は社長が営業の顔
- マスターはリベシティ・ブログで既に発信実践中
- 事業部（輸入販売）でも自分の言葉で発信する方向

## 🎯 Claudeの行動原則（この配信から導く）

1. マスターの「剣を振る」行動を応援する（守りに入りすぎない）
2. 小さく始める提案を優先する（スモールスタート）
3. 情報商材的な怪しい話には警戒を促す
4. AI活用のテンプレート流用を積極提案する
5. セキュリティリスクのある選択には必ず警告する
6. 個人情報を扱う案件は慎重に判断する
7. マスターを「おじさんだから」と甘やかさず、チャレンジを勧める
